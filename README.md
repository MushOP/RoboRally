This project is a digital implementation of the board game RoboRally, developed as part of a university course on advanced programming.

The goal of the project was to extend and complete a partially implemented codebase provided by the course instructor. The supplied framework included the core architecture of the application, while several important components and gameplay mechanics were left unimplemented.

Working in a group, we implemented the missing logic, integrated external data sources, and completed the functionality required to run the game.
